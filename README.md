# 💊 PharmaTrace
### Drug Interaction Network Analyzer

PharmaTrace maps drug interactions as a network and finds the most dangerous
interaction path between any two medications — using real FDA adverse event data.

Built for COP3530 · Data Structures and Algorithms · University of Florida

---

## What It Does

You enter two drug names. PharmaTrace searches a network of 6,790 drugs and
417,801 known interactions to find the riskiest connection between them.

It runs two algorithms side by side and tells you which one was faster:

```
Enter Drug 1: WARFARIN
Enter Drug 2: ASPIRIN

Dijkstra    →  Path found in 237 ms   · 835 nodes visited
Bellman-Ford →  Path found in 2402 ms  · 3,917,899 nodes visited

⛔ AVOID: High interaction risk (0.87)
   Do not take together without direct physician supervision.
```

---

## How the Data Works

The interaction network is built from the **FDA Adverse Event Reporting System (FAERS)** —
a public database of real patient adverse event reports submitted to the FDA.

```
Patient report:  took Drug A + Drug B + Drug C → bad reaction
                          ↓
         We create edges:  A──B,  A──C,  B──C
                          ↓
         Edge weight  =  severity × log(how often they co-occur)
```

The more often two drugs appear together in serious adverse events,
the higher their interaction risk score.

**Dataset:**  FDA FAERS Q2 2025  ·  393,130 reports  ·  1.8M drug records

---

## The Two Algorithms

Both algorithms solve the same problem: find the highest-risk path between two drugs.
We compare them to show the real-world performance difference.

| | Dijkstra | Bellman-Ford |
|---|---|---|
| Strategy | Explores closest nodes first using a min-heap | Relaxes every edge repeatedly |
| Time Complexity | O((V + E) log V) | O(V × E) |
| Speed | Fast | ~5x slower |
| Negative cycles | Cannot detect | ✓ Can detect |
| Best for | Most queries | Graphs with negative weights |

Both are implemented **from scratch** — no algorithm libraries used.

## Getting Started

**Requirements:** C++17, Python 3, pandas, numpy

**Step 1 — Clone the repo**
```bash
git clone https://github.com/OmarSayeh/PharmaTrace.git
cd PharmaTrace
```

**Step 2 — Download FAERS data**

Download the Q2 2025 ASCII files from the FDA:
```
https://fis.fda.gov/extensions/FPD-QDE-FAERS/FPD-QDE-FAERS.html
```
Place `DRUG25Q2.txt` and `REAC25Q2.txt` in the `data/` folder.

**Step 3 — Generate the interaction network**
```bash
python preprocessing.py
```
This creates `data/edges.csv` — the full interaction graph (~2 min).

**Step 4 — Compile**
```bash
clang++ -std=c++17 src/main.cpp src/graph.cpp src/dijkstra.cpp src/bellmanford.cpp src/menu.cpp -o rxpath
```

**Step 5 — Run**
```bash
./rxpath
```
